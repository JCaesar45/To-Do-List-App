## How it Works

- Users can input a task and click "Add" or press Enter.
- Tasks appear below with options to mark as completed or delete.
- Completed tasks are crossed out with a different style.
- The interface is styled with Facebook's color palette to give a familiar look.

## Customization

- Change the theme colors by editing the CSS variables in `style.css`.
- Enhance functionality, such as editing tasks or saving to local storage.
